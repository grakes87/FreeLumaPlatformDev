**Video background layer** — same pattern as DailyPostSlide:
- Dark base: `<div className="absolute inset-0 bg-[#0a0a0f]" />`
- `<video>` element with `ref={bgVideoRef}`, `src={content.video_background_url}`, `crossOrigin="anonymous"`, `autoPlay`, `muted`, `loop`, `playsInline`, `preload="none"` (use `preload="none"` NOT `preload="auto"` — the verse slide already preloads this same URL, so the browser cache will have it; this avoids double-downloading on mobile)

- Fade-in transition: video has `opacity-0` until ready, then `opacity-100` with `transition-opacity duration-500`

**isActive media isolation** — same pattern as DailyPostSlide:
```typescript
useEffect(() => {
  const video = bgVideoRef.current;
  if (!video) return;
  if (isActive) {
    video.play().catch(() => {});
  } else {
    video.pause();
  }
}, [isActive]);
```

**Gradient overlay:**
- `<div className="absolute inset-0 bg-gradient-to-t from-black/80 via-black/50 to-black/40" />`
- Same as DailyPostSlide for consistent look

**Content overlay** — centered devotional text with share button at bottom:
- Full-height flex column with same padding as DailyPostSlide:
  - `paddingTop: 'calc(3.5rem + env(safe-area-inset-top, 0px) + 0.5rem)'`
  - `paddingBottom: 'calc(4rem + env(safe-area-inset-bottom, 0px) + 2.5rem)'`

**Top section:** Small label — "Devotional Reflection" in uppercase tracking-widest, matching the style of the "Bible Verse" / "Daily Quote" labels on the verse slide:
```jsx
<div className="text-center">
  <p className="text-xs font-medium uppercase tracking-widest text-white/60">
    Devotional Reflection
  </p>
</div>
```

**Center section:** Devotional reflection text. Key design decisions:
- Use `font-sans` (NOT italic serif like the verse slide — per research, this visually distinguishes reflection from verse quotes)
- Text styling: `text-lg leading-relaxed font-light text-white drop-shadow-lg sm:text-xl md:text-2xl`
- Use dynamic font sizing based on text length for readability:
  - Short text (< 200 chars): `text-xl sm:text-2xl`
  - Medium text (200-500 chars): `text-lg sm:text-xl`
  - Long text (500+ chars): `text-base sm:text-lg`
- Max width container: `max-w-lg`
- Center aligned text
- Text shadow for readability: `style={{ textShadow: '0 2px 8px rgba(0,0,0,0.5)' }}`
- If text is very long (800+ chars), add `overflow-y-auto max-h-[60vh]` as a safety net with a subtle fade at the bottom

**Bottom section:** ShareButton only (per user decision — no reactions, no comments):

  />
